# TGS2024 出展作品

## 概要
![Game Thumbnail](./images/poster.png)

実際のバーコードリーダーを入力デバイスとして使用する、Unity製リアクションゲームです。  
プレイヤーはランダムに出現する二人の配達員の要求を判断し、  
対応する商品のバーコードをスキャンすることでスコアを獲得します。

本作品は、**実機デバイスを用いた入力処理の実装**と、  
**チーム開発における役割分担・コミュニケーションを含む制作プロセスの実践**を目的として制作しました。
